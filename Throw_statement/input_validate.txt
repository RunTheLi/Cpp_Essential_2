Objectives

adding code to handle exceptions;
adding code to throw exceptions.
Scenario
Try to change the program below (both the div and main functions).

Insert a throw and a try-catch block instead of a Boolean value and res parameter (change it to a function return value).

Remember to inform the user of any exceptions in simple words.

Try to make two versions of your code, one that catches the division by zero exception, and another that checks arguments and then throws an exception (when the argument isn't proper). Which version is better?

Sample Input

1
Expected output
2

Sample Input
6
0
Expected output
Are you kidding me?
Your input is not valid. You can't divide by zero.

// Base Code //
#include <iostream>

using namespace std;
const int DivideByZero = 111;
using namespace std;
float internaldiv(float arg1, float arg2)
{
	if (0==arg2)	
		throw DivideByZero;
	return arg1 / arg2;
}

bool div(float &res, float arg1, float arg2) {
	if(arg2 == 0.0)
		return false;
	internaldiv(arg1, arg2);
	return true;
}
